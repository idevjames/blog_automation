# 🤖 네이버 블로그 자동화 봇 (v1.0)

블로그 활동을 사람처럼 자연스럽게 수행하는 자동화 프로그램입니다.  
단순 반복 매크로와 달리 랜덤 딜레이, 실제 마우스 좌표 클릭, 요소 탐색 등을 통해 **실제 사용자의 행동 패턴**을 모방하여 작동합니다.

---

## ⚙️ 설정 방법 (공통)

1. 프로그램 실행 전, 루트 폴더의 **`settings.txt`** 파일을 메모장으로 열어 환경을 설정하세요.
   - **NEIGHBOR_MESSAGES**: 서로이웃 신청 시 보낼 랜덤 메시지 목록
   - **CUSTOM_DELAYS**: 사람처럼 보이기 위한 행동별 대기 시간 (초 단위)
   - **실패 기준 설정**:
     - **DEFAULT_LIKE_FAILURE_COUNT**: 공감 시도 연속 실패 시 중단 기준 (기본 5회)
     - **DEFAULT_ADD_NEIGHBOR_FAILURE_COUNT**: 서로이웃 신청 연속 실패 시 중단 기준 (기본 5회)

---

## 🪟 Windows (윈도우) 사용법

윈도우 사용자는 복잡한 명령어 입력 없이 **`run.bat`** 파일만 실행하면 됩니다.

### 1. 실행 방법
- 폴더 내의 **`run.bat`** 파일을 우클릭하여 **`관리자권한으로 실행`**을 선택합니다.

### 2. 파이썬 자동 설치 (Python이 없는 경우)
- 컴퓨터에 파이썬이 설치되어 있지 않다면, 자동으로 설치 안내가 뜨고 진행됩니다.
- **⚠️ 중요**: 자동 설치가 완료되면 설정 적용을 위해 창이 닫힙니다. **반드시 다시 한번 `run.bat`를 더블클릭**해야 봇이 정상 실행됩니다.
- 만약 정상적으로 되지 않는다면 수동으로 설치하셔야합니다

### 3. 참고 사항
- 한글 깨짐 방지를 위해 실행창의 모든 안내 메시지는 **영어(English)**로 출력됩니다.
- `[START] Running the bot...` 메시지가 뜨면 정상적으로 실행된 것입니다.
- 최초 실행 시 가상환경(venv) 생성 및 패키지 설치로 인해 시간이 조금 걸릴 수 있습니다.

## 💡 주요 기능 및 작동 로직

##### 이웃 새글 공감하기
**[기능]** 
- 이웃들의 새 글 목록을 순회하며 공감(하트)을 누릅니다.
**[로직]**
- `aria-pressed` 속성을 확인하여 이미 공감을 누른 글은 스마트하게 패스합니다.
**[안전 장치]**
- 공감 버튼을 찾지 못하거나 클릭 오류가 **5회 연속** 발생하면 작업을 중단합니다.

##### 주제별 서로이웃 신청하기
**[기능]**
- 설정한 관심사(`settings.txt`)의 블로그를 찾아 서로이웃을 신청합니다.

**[로직]** 
  - 이미 서로이웃이거나 신청 버튼이 없는 경우 건너뜁니다.
  - 팝업 창에서 '서로이웃' 선택 -> 메시지 입력 -> 신청 완료까지의 과정을 단계별로 검증합니다.
**[안전 장치]**: "신청을 완료했습니다" 메시지를 확인하지 못하는 실패가 **5회 연속** 발생하면, 봇 보호를 위해 작업을 조기 종료합니다.

##### 자동 로그인 (세션 유지)
- 최초 1회 로그인 시 `system/naver_profile` 폴더에 로그인 정보를 저장합니다.
- 로그인 유지 버튼을 선택하고 로그인하면 이후 실행 시에는 별도의 로그인 과정 없이 **자동으로 로그인 상태가 유지**됩니다.
