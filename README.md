# 🤖 네이버 블로그 자동화 봇 (v1.0)

블로그 활동을 사람처럼 자연스럽게 수행하는 자동화 프로그램입니다.  
단순 반복 매크로와 달리 랜덤 딜레이, 실제 마우스 좌표 클릭, 요소 탐색 등을 통해 **실제 사용자의 행동 패턴**을 모방하여 작동합니다.

---

## ⚙️ 설치 방법 (공통)
1. code를 누른 후 download zip을 눌러서 압축파일을 다운로드 받는다
2. 다운로드 받은 파일을 압축해제한다.
3. 아래 OS에 맞게 (윈도우/mac) 진행한다.
4. 실행한 프로그램에서 지시하는대로 진행한다

---

## 🪟 Windows (윈도우) 사용법
### 1. 실행 방법
- 폴더를 열어 **`run (확장자 bat)`** 파일을 우클릭하여 **`관리자권한으로 실행`**을 선택합니다.

### 2. 파이썬 자동 설치 (Python이 없는 경우)
- 컴퓨터에 파이썬이 설치되어 있지 않다면, 자동으로 설치 안내가 뜨고 진행됩니다.
- **⚠️ 중요**: 자동 설치가 완료되면 설정 적용을 위해 창이 닫힙니다. **반드시 다시 한번 `run.bat`를 더블클릭**해야 봇이 정상 실행됩니다.
- 만약 정상적으로 되지 않는다면 수동으로 설치하셔야합니다 (준행이한테 문의)

### 3. 참고 사항
- 한글 깨짐 방지를 위해 실행창의 모든 안내 메시지는 **영어(English)**로 출력됩니다.
- `[START] Running the bot...` 메시지가 뜨면 정상적으로 실행된 것입니다.
- 최초 실행 시 가상환경(venv) 생성 및 패키지 설치로 인해 시간이 조금 걸릴 수 있습니다.

## 🍎 macOS (맥) 사용법

맥의 보안 정책상 `run.command`를 더블클릭하면 "확인되지 않은 개발자" 또는 "악성 코드" 경고가 뜨며 실행되지 않습니다. 따라서 **터미널(Terminal)**을 통해 실행해야 합니다.

### 1. 터미널 열기 및 폴더 이동
- `Command + Space`를 눌러 **'터미널'**을 검색해 실행합니다.
- `cd `를 입력하고 한 칸 띄운 뒤, 프로그램 폴더를 터미널 창으로 드래그 앤 드롭 하거나 직접 경로를 입력합니다.
  ```bash
  # 예시: 바탕화면에 폴더가 있는 경우
  cd Desktop/blog_automation-main
1. 프로그램 실행 전, **settings.txt** 파일을 메모장으로 열어 환경을 설정하세요.
   - **NEIGHBOR_MESSAGES**: 서로이웃 신청 시 보낼 랜덤 메시지 목록
   - **CUSTOM_DELAYS**: 사람처럼 보이기 위한 행동별 대기 시간 (초 단위)
   - **실패 기준 설정**:
     - **DEFAULT_LIKE_FAILURE_COUNT**: 공감 시도 연속 실패 시 중단 기준 (기본 5회)
     - **DEFAULT_ADD_NEIGHBOR_FAILURE_COUNT**: 서로이웃 신청 연속 실패 시 중단 기준 (기본 5회)
