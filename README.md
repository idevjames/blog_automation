# 🤖 네이버 블로그 자동화 봇

본 프로그램은 실제 사용자의 행동 패턴을 모방하여 블로그 활동을 자연스럽게 수행하는 **GUI 기반 자동화 솔루션**입니다. 별도의 코딩이나 복잡한 환경 설정 없이, 빌드된 실행 파일을 통해 직관적으로 이웃 소통 및 서로이웃 신청 작업을 관리할 수 있습니다.

---

## 📥 설치 및 실행 방법 (OS별)

### 1. 최신 버전 다운로드
1. 저장소의 우측의 **[Releases]** 페이지에서 최신 버전을 누른다.
<img width="1225" height="522" alt="스크린샷 2026-01-13 오후 10 47 00" src="https://github.com/user-attachments/assets/b85535cc-0ad6-408e-868f-28389caef2f8" />

2. 본인의 운영체제에 맞는 압축 파일을 다운로드합니다.
  * **Windows**: `blog_automation_windows.zip`
  * **macOS**: `blog_automation_mac.zip`
<img width="975" height="459" alt="스크린샷 2026-01-13 오후 10 48 41" src="https://github.com/user-attachments/assets/c2b2288c-8cbe-4be3-9167-2a2b9f66ae59" />



### 2. 실행 방법
* **Windows (윈도우)**:
  1. 다운로드한 압축 파일을 해제합니다.
  2. 폴더 내 **`gui_main.exe`** 파일을 실행합니다.
  3. (최초 실행 시 PC 보호 팝업이 뜨면 `추가 정보` -> `실행`을 클릭하세요.)
* **macOS (맥)**:
  1. 압축 해제 후 **`gui_main.app`** 파일을 실행합니다.
  2. 시스템 보안 정책으로 실행이 차단될 경우, `우클릭 > 열기`를 선택하거나 `시스템 설정 > 개인정보 보호 및 보안`에서 실행을 허용해 주세요.
