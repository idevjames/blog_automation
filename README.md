# 🤖 네이버 블로그 자동화 봇 v2.0 (GUI Release)

본 프로그램은 실제 사용자의 행동 패턴을 모방하여 블로그 활동을 자연스럽게 수행하는 **GUI 기반 자동화 솔루션**입니다. 별도의 코딩이나 복잡한 환경 설정 없이, 빌드된 실행 파일을 통해 직관적으로 이웃 소통 및 서로이웃 신청 작업을 관리할 수 있습니다.

## 🖥️ 주요 기능 및 UI 가이드

프로그램 실행 시 나타나는 GUI 창을 통해 모든 기능을 제어하며, 로그를 실시간으로 확인할 수 있습니다.

### 1. 로그인
* 네이버 창이 열리면 로그인 시 `[로그인 상태 유지]` 를 체크해주셔야 로그인 기록이 저장됩니다.
* 만약 빈창이 뜨고 실행이 잘 안되면 크롬을 모두 닫고 우측 상단의 `[브라우저 재실행]` 버튼을 눌러주세요 (그래도 안되면 프로그램 재실행)

### 1. ❤️ 이웃 새글 공감 (탭 1)
* **목표 설정**: 공감을 남길 목표 포스팅 개수를 지정합니다.
* **시작 페이지**: 탐색을 시작할 페이지 번호를 설정합니다.
* **수치 저장**: 하단 설정 영역에서 딜레이와 실패 횟수 등을 수정한 후 **[💾 공감 수치 설정 저장]**을 누르면 `게시글공감설정.txt`에 즉시 반영됩니다.

### 2. 🤝 서로이웃 신청 (탭 2)
* **주제별 탐색**: 네이버 주제별 카테고리(대분류/상세주제)를 선택하여 타겟 블로거를 자동으로 추출합니다.
* **조건 필터링**: 공감 수나 댓글 수가 설정값 이하인 블로그만 방문하도록 필터링하여 '진짜 소통 이웃'을 찾습니다.
* **메시지/댓글 편집**: UI 내 **[📂 서이추 메시지 열기]**, **[📂 댓글 관리 열기]** 버튼을 통해 관련 텍스트 파일을 즉시 편집할 수 있습니다.
